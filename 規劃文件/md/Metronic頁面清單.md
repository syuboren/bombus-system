# Metronic 模版 HTML 頁面分析清單

本文件提供了 `metronic/dist` 目錄下可用的 HTML 頁面完整清單，並依據其功能進行分類。

## 1. 儀表板與登陸頁 (Dashboards & Landing)
針對不同商業領域的核心儀表板佈局。

*   `index.html` - **多用途儀表板 (Multipurpose Dashboard)** (主要入口)
*   `landing.html` - **登陸頁面 (Landing Page)**
*   `dashboards/ecommerce.html` - 電子商務儀表板 (eCommerce Dashboard)
*   `dashboards/marketing.html` - 行銷儀表板 (Marketing Dashboard)
*   `dashboards/logistics.html` - 物流儀表板 (Logistics Dashboard)
*   `dashboards/delivery.html` - 配送儀表板 (Delivery Dashboard)
*   `dashboards/online-courses.html` - 線上課程儀表板 (Online Courses Dashboard)
*   `dashboards/website-analytics.html` - 網站分析儀表板 (Website Analytics Dashboard)
*   `dashboards/finance-performance.html` - 財務績效儀表板 (Finance Performance Dashboard)
*   `dashboards/store-analytics.html` - 商店分析儀表板 (Store Analytics Dashboard)
*   `dashboards/social.html` - 社群媒體儀表板 (Social Media Dashboard)
*   `dashboards/crypto.html` - 加密貨幣儀表板 (Cryptocurrency Dashboard)
*   `dashboards/school.html` - 學校/教育儀表板 (School/Education Dashboard)
*   `dashboards/bidding.html` - 競標/拍賣儀表板 (Bidding/Auction Dashboard)
