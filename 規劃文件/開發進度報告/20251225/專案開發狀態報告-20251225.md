## 🏗️ 系統架構

```
src/app/
├── core/                    # 核心服務 (Guards, Interceptors)
│   ├── guards/
│   ├── interceptors/
│   └── services/
│       ├── notification.service.ts
│       └── sidebar.service.ts
├── shared/                  # 共用元件
│   ├── animations/
│   ├── components/
│   │   ├── circular-progress/
│   │   ├── header/
│   │   ├── notification/
│   │   ├── sidebar/
│   │   └── stat-card/
│   ├── directives/
│   └── pipes/
└── features/                # 功能模組
    ├── auth/                # 身份驗證
    ├── dashboard/           # CEO 儀表板
    ├── employee/            # 員工管理
    ├── training/            # 培訓與人才發展
    ├── project/             # 專案管理
    ├── competency/          # 職能管理
    └── organization/        # 組織架構
```

## 🛠️ 技術棧

### 前端框架
- **Angular 18.2** - Standalone Components 架構
- **TypeScript 5.5**
- **RxJS 7.8**

### UI/UX
- **SCSS** - 自定義設計系統與變數
- **Bento UI Layout** - 現代化儀表板布局
- **Remix Icon** - 圖示庫
- **Responsive Design** - 完整響應式設計

### 資料視覺化
- **ECharts 6.0** - 圖表繪製
- **ngx-echarts 18.0** - Angular ECharts 整合

### 文件匯出
- **html2canvas 1.4** - 畫面截圖
- **jspdf 3.0** - PDF 生成
- **jspdf-autotable 5.0** - PDF 表格

### 部署
- **GitHub Pages** - 靜態網站託管
- **angular-cli-ghpages 2.0** - 自動部署工具

## 🎯 本次儀表板重點更新

### 1. 快速導航 (Quick Nav)
- 右側隱藏式快速導航選單
- 支援收合/展開功能
- 平滑滾動至各區塊
- 自動偵測當前區塊高亮

### 2. 區塊分組標題
- **人才管理** (Row 2-5) - 👥 綠色系
- **專案交付** (Row 6-7) - 📊 橙色系
- **毛利損益** (Row 8) - 💰 藍綠色系
- **文化訊號** (Row 9) - 💜 紫色系
- **決策事項** (Row 10) - ✅ 警示色系

### 3. 專案進度追蹤 (甘特圖)
- 甘特圖顯示專案時程
- 依專案類型顯示不同顏色
- 點擊顯示專案詳細資訊

### 4. 成本結構分析 (Bullet Chart)
- 四項成本類別子彈圖
- 估算 vs 實際成本比較
- Hover 顯示金額數值

### 5. 績效薪資優化
- 新增調薪公平性環狀圖
- 統一「獎酬」為「薪資」用詞
