# 職等職級矩陣 (Grade Matrix) 系統實作文件

本文件記錄 `competency/grade-matrix` 功能的技術實作細節，包括程式碼位置、資料庫結構與欄位對照。

## 1. 程式碼位置 (Codebase Locations)

### 前端 (Frontend)
- **頁面組件**: `src/app/features/competency/pages/grade-matrix-page/grade-matrix-page.component.ts`
- **頁面模板**: `src/app/features/competency/pages/grade-matrix-page/grade-matrix-page.component.html`
- **資料服務**: `src/app/features/competency/services/competency.service.ts`

### 後端 (Backend)
- **API 路由**: `server/src/routes/grade-matrix.js`
- **資料庫檔案**: `server/src/db/onboarding.db` (SQLite)
- **資料庫初始化**: `server/src/db/init.js`

---

## 2. 資料庫結構與欄位對照 (Database Schema & Fields)

### 2.1 職等基本資料表 (`grade_levels`)
儲存 7 個職等的定義。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | ID | TEXT | 唯一識別碼 | `grade-1` |
| `grade` | 職等 | INTEGER | 數字職等 (1-7) | `1` |
| `code_range` | 職級範圍 | TEXT | 對應的職級代碼範圍 | `BS01` |
| `title_management` | 管理職稱 | TEXT | 管理職軌道的名稱 | `新人級` |
| `title_professional` | 專業職稱 | TEXT | 專業職軌道的名稱 | `新人級` |
| `education_requirement` | 學歷與資格 | TEXT | 該職等的基本要求 | `符合職位學歷與經歷條件...` |
| `responsibility_description` | 職責說明 | TEXT | 主要職責描述 | `新進人員學習與適應...` |

### 2.2 職級薪資表 (`grade_salary_levels`)
儲存每個職級的薪資標準。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `grade` | 所屬職等 | INTEGER | 關聯的職等 | `1` |
| `code` | 職級代碼 | TEXT | 詳細職級代碼 | `BS01` |
| `salary` | 薪資 | INTEGER | 標準薪資 | `35000` |
| `sort_order` | 排序 | INTEGER | 顯示順序 | `1` |

### 2.3 部門職位配置表 (`department_positions`)
定義各部門在每個職等的具體職稱。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `department` | 部門名稱 | TEXT | 所屬部門 | `執行長辦公室` |
| `grade` | 職等 | INTEGER | 對應職等 | `7` |
| `title` | 職稱 | TEXT | 具體職稱 | `執行長` |
| `track` | 軌道類型 | TEXT | `management` / `professional` | `management` |
| `supervised_departments` | 管轄部門 | TEXT | (JSON) 跨部門主管管轄範圍 | `["執行長辦公室","行政部"...]` |

### 2.4 職涯發展路徑表 (`career_paths`)
定義職涯發展的標準路徑。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | ID | TEXT | 路徑 ID | `path-vertical-mgmt` |
| `type` | 類型 | TEXT | `vertical`, `horizontal`, `cross` | `vertical` |
| `name` | 名稱 | TEXT | 路徑名稱 | `管理職垂直晉升` |
| `description` | 描述 | TEXT | 路徑說明 | `從新人級一路晉升...` |
| `from_position` | 起點 | TEXT | 起始職位 | `新人級 (Grade 1)` |
| `to_position` | 目標 | TEXT | 目標職位 | `營運管理 (Grade 7)` |
| `estimated_time` | 預估時間 | TEXT | 完成所需時間 | `10-15年` |
| `steps` | 階段 | TEXT | (JSON) 詳細發展階段 | (見下方 JSON) |

## 3. API 介面 (API Endpoints)

| HTTP 方法 | 路徑 | 說明 | 備註 |
| :--- | :--- | :--- | :--- |
| `GET` | `/api/grade-matrix` | 匯總矩陣資料 | 包含職等與薪資資訊 |
| `GET` | `/api/grade-matrix/departments/list` | 部門列表 | |
| `GET` | `/api/grade-matrix/positions/list` | 部門職位對照 | 支援 `department`, `grade`, `track` 篩選 |
| `GET` | `/api/grade-matrix/career/paths` | 職涯路徑 | 支援 `type` 篩選 |
| `GET` | `/api/grade-matrix/:grade` | 單一職等詳情 | 包含該職等的晉升條件 |
