## 2. 資料庫結構與欄位對照 (Database Schema & Fields)

### 2.1 職能主表 (`competencies`)
儲存所有類型的職能基本資訊 (Core, Management, Professional, KSA)。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | ID | TEXT | 唯一識別碼 | `core-communication` |
| `code` | 代碼 | TEXT | 職能代碼 | `CORE-01` |
| `name` | 名稱 | TEXT | 職能名稱 | `溝通表達` |
| `type` | 類型 | TEXT | `core`, `management`, `professional`, `ksa` | `core` |
| `category` | 類別 | TEXT | 同 `type` (目前與 type 重複) | `core` |
| `description` | 描述 | TEXT | 職能定義與說明 | `能夠清晰、準確地傳達資訊...` |
| `created_at` | 建立時間 | TEXT | ISO 8601 格式 | `2026-02-02T12:08:08.226399` |
| `updated_at` | 更新時間 | TEXT | ISO 8601 格式 | `2026-02-09T04:19:40.794Z` |

### 2.2 職能等級表 (`competency_levels`)
儲存分級職能 (Core, Management, Professional) 的各等級行為指標。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | ID | TEXT | 唯一識別碼 | `75eb0c41-8c29...` |
| `competency_id` | 職能 ID | TEXT | 關聯 `competencies.id` | `core-problem-solving` |
| `level` | 等級 | TEXT | `L1` - `L6` | `L1` |
| `indicators` | 行為指標 | TEXT | JSON 陣列，儲存該等級的具體行為 | `["辨識作業異常...", "運用基礎工具...", "執行指派任務..."]` |

### 2.3 KSA 詳細資訊表 (`competency_ksa_details`)
儲存 KSA 類型職能的特有欄位 (無等級區分)。

| 欄位名稱 (English) | 欄位名稱 (Chinese) | 類型 | 說明 | 範例資料 |
| :--- | :--- | :--- | :--- | :--- |
| `id` | ID | TEXT | 唯一識別碼 | `64a832c2-8dc8...` |
| `competency_id` | 職能 ID | TEXT | 關聯 `competencies.id` | `ksa-k-fin-01` |
| `behavior_indicators` | 行為指標 | TEXT | JSON 陣列 | `["了解並正確應用國際會計準則", "熟悉商業會計法規"...]` |
| `linked_courses` | 關聯課程 | TEXT | JSON 陣列 | `[]` |

## 3. API 介面 (API Endpoints)

| HTTP 方法 | 路徑 | 說明 | 備註 |
| :--- | :--- | :--- | :--- |
| `GET` | `/api/competencies` | 取得職能列表 | 支援 `type`, `category` 篩選參數 |
| `GET` | `/api/competencies/stats` | 取得職能統計 | 回傳各類別數量統計 |
| `GET` | `/api/competencies/:id` | 取得單一職能詳情 | 包含 Levels 或 KSA Details |
| `DELETE`| `/api/competencies/:id` | 刪除職能 | (需確認是否實作) |

## 4. 實作備註 (Implementation Notes)

1.  **資料結構差異**:
    *   **Core / Management / Professional**: 採用分級制 (Level-based)，關聯 `competency_levels` 表，每個職能有 L1-L6 六個等級。
    *   **KSA (Knowledge, Skill, Attitude)**: 不分等級，僅有關聯課程與行為指標，關聯 `competency_ksa_details` 表。

2.  **前端展示邏輯**:
    *   Core/Management/Professional 使用 `Hierarchy` (階層) 顯示方式，可展開查看各等級指標。
    *   KSA 使用 `Card` (卡片) 或 `List` (列表) 顯示方式，支援搜尋與篩選。

3.  **顏色代碼**:
    *   **Core (核心)**: `#D6A28C` (陶土橙)
    *   **Management (管理)**: `#9A8C98` (錦葵紫)
    *   **Professional (專業)**: `#4A7C6F` (深綠)
    *   **KSA**: `#8DA399` (鼠尾草綠)
